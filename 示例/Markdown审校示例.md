# Markdown 审校输出示例

## 待审校片段

```markdown
本方案新增批量读取能力。通过 Agent 对 memory modules 进行读取，实现用户信息的结构化注入。
```

## 审校建议

**总体判断**

这段话能看出方案方向，但动作句不够直接，中英混用较多，业务读者需要多读几遍才能理解。建议先说明本方案新增什么能力，再解释关键术语和实际结果。

**高优先级问题**

- `memory modules` 没有中文解释，混合读者不一定知道它指什么。
- “结构化注入”偏工程黑话，可以改成更具体的动作。

**可替换片段**

```markdown
本方案新增批量读取能力，Agent 按需读取指定的记忆模块 `memory modules`，并把用户信息整理成模型可直接使用的上下文。
```

**仍需确认**

- `memory modules` 是否是正式字段名。
- `memory modules` 的读取范围是否已有统一口径。
